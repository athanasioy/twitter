### Tweet Data Saving
**For Tweets**
We will save
- Tweet ID
- Tweet Text
- Tweet creation data
- Author ID
- Retweet ID (referenced_tweets.id)
- AnswersTo (in_reply_to_user_id)
- Source
**For Users**
- User ID
- name
- Username
- Creation Date
